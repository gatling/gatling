#######
Gatling
#######

Gatling is a highly capable load testing tool. It is designed for easy of use, maintainability and high performance. Out of the box, Gatling comes with an excellent support of the HTTP protocol and makes it a tool of choice for load testing any HTTP server. Moreover, most of its code is protocol agnostice, simplifying the implementation of new protocols.


The :doc:`quickstart <Quickstart>` has an overview of the most important concepts, walking you through the setup of a simple scenario for load testing an HTTP server.
